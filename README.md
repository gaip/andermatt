# IT Transformation Roadmap - Andermatt Swiss Alps AG

This project contains the 90-day transformation roadmap for Andermatt Swiss Alps AG.

## Live Website
**[https://andermatt-roadmap.azurewebsites.net](https://andermatt-roadmap.azurewebsites.net)**

## Local Review
1.  **Right-click `index.html`** in Android Studio and select **Open in > Browser**.
2.  Alternatively, run:
    ```bash
    npm install
    npm start
    ```
    Then visit `http://localhost:3000`.

## Deployment (Azure)
This project is hosted on **Azure App Service**.

### One-Click Deploy
To push updates to the live site, run:
```bash
npm run deploy-azure
```
